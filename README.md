# Missile Defense Simulation

## ⚠️ Disclaimer

This project is created **solely for educational and experimental purposes** as part of AI Vibe coding demonstrations. It is intended to showcase 3D simulation techniques, physics modeling, and interactive visualization using React Three Fiber.

This simulation is **not intended for any real-world defense applications** and should be used only for learning and experimentation.

---

## About

A full-featured 3D missile defense simulation built with React, Three.js, and React Three Fiber. Features include:

- Real-time 3D visualization
- Physics-based missile trajectories
- Iron Dome defense system simulation
- Interactive camera controls
- Missile POV camera mode

## Tech Stack

- React 18
- Three.js
- @react-three/fiber
- @react-three/drei
- Zustand (state management)
- Vite (build tool)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Controls

- 🖱️ Left Click: Rotate camera
- 🖱️ Right Click: Pan camera
- 🖱️ Scroll: Zoom
- ⌨️ Tab: Switch camera views
